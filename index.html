Architecture of an Azure-based web application system

User requests go through a load balancer to web servers.

Requests are routed via Azure Application Gateway to Azure Functions.

Azure Functions handle the logic and interact with:

  • Azure Storage (files)
  • SQL Database (structured data)
  • Cosmos DB (NoSQL data)

Data may be passed to downstream systems for further processing.
